Unity Meta Files Automatic Compression

Source Code (GitHub)

This Unity plugin automatically compresses all ".meta" files into a single ".zip" file. This allows you to add ".meta" files to the ".gitignore", making Unity game repositories on GitHub easier to navigate by reducing the number of files in folders. On the first launch of the project, the plugin will automatically unzip the meta files, if only MetaFilesCompressed.zip file exist in your project.
